# Visual Assets for "Everyone Wants AI Coding Agents Until Security Joins the Standup"

## Core Creative Direction

Humorous but credible enterprise-tech visual. The central idea is an eager AI coding agent colliding with real-world security and governance controls. The image should feel sharp, modern, and slightly satirical rather than cartoonish.

## Thumbnail Prompt

Create a polished editorial-style LinkedIn thumbnail showing a glowing AI coding assistant at a developer workstation, confidently generating code on multiple monitors, while a serious security lead joins a standup holding a clipboard labeled permissions, audit trail, secrets, and compliance. On the screens, subtle UI hints show pull requests, CI checks, and red access-denied warnings. Tone is witty, high-trust, enterprise software aesthetic, cinematic lighting, clean composition, blue-gray palette with orange warning accents, realistic illustration, no brand logos, no watermarks, minimal text.

## Social Graphic Prompt

Design a modern social graphic for LinkedIn about AI coding agents meeting enterprise reality. Split-scene composition: left side shows smooth AI-assisted coding flow with green checks, fast commits, and documentation generation; right side shows security review gates, policy prompts, secret vault icons, branch protections, and governance checkpoints. Style is premium B2B tech campaign, intelligent humor, crisp UI-inspired overlays, realistic-digital hybrid artwork, highly legible focal point, balanced negative space for optional headline placement, no logos, no clutter.

## Optional Overlay Copy

- Everyone wants AI agents
- Until security joins the standup
- Fast code, slower trust

## Visual Notes

- Favor realism over memes.
- Keep humor in the scenario, not exaggerated facial expressions.
- Avoid dystopian robot imagery.
- Emphasize trust, controls, and modern developer workflow.
