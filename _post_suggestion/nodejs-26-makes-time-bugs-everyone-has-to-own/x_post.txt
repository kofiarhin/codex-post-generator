Node.js 26 makes JavaScript time bugs harder to hand-wave.

Temporal being enabled by default is not a glamorous runtime feature. It is a quiet admission that `Date` made too many serious systems rely on vibes, wrapper functions, and heroic comments near billing code.

The hard part is not learning a new API.

The hard part is finding every place where timezone math, serialization, scheduled jobs, and product rules got blurred together because the old tools were too awkward to model reality cleanly.

Useful upgrade. Uncomfortable migration. Very normal software.
